# 🚀 Astro OBS

O **Astro OBS** é uma plataforma focada em astronomia, projetada para conectar entusiastas do espaço com os eventos astronômicos mais importantes do momento. O projeto combina design moderno (estilo *Cosmic Deep Space*), usabilidade focada em dispositivos móveis e integração com dados astronômicos.

## 🌌 Sobre o Projeto
Este projeto foi desenvolvido como parte da transição para uma carreira como desenvolvedor front-end autônomo para o curso de HTML e CSS do OXETECH. O objetivo foi criar uma interface que unisse estética tecnológica, navegação intuitiva e funcionalidade, permitindo ao usuário se manter atualizado sobre o cosmos.

## 🛠 Tecnologias Utilizadas
- **HTML5:** Estruturação semântica do conteúdo.
- **CSS3:** Estilização responsiva, utilizando Grid/Flexbox, animações personalizadas e variáveis CSS para gerenciamento de temas.
- **JavaScript (ES6):** Manipulação do DOM, implementação de scroll suave e lógica de interatividade.
- **Google Fonts:** Tipografia hierárquica (Orbitron e Montserrat).
- **GitHub Pages:** Deploy e hospedagem do projeto.

## 🚀 Funcionalidades
- **Interface Responsiva:** Layout adaptável para telas mobile e desktop.
- **Design Temático:** Identidade visual inspirada em tecnologia espacial, com efeitos de brilho e transparência (*Glassmorphism*).
- **Navegação Inteligente:** Menu fixo, scroll suave entre seções e botão flutuante interativo de conversão.
- **Sistema de Notícias:** Páginas dedicadas para artigos astronômicos, com links externos para fontes oficiais.
- **Performance:** Otimização de carregamento de recursos e organização de arquivos.
